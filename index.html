Yelp Review AI Analysis

Comparative sentiment analysis across 50 restaurant reviews using custom GPT models. Qualitative data → operational recommendations.

Built a custom sentiment taxonomy, coded 25 reviews each for Lucky Robot and Cabo Bob's. Both showed ~36% positive compliments but differed in how they used — or ignored — negative signals. Delivered operational recommendations based on pattern analysis.
